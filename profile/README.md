## Overview

<a href="#">Coda 2</a> combines web development tools into a single, beautifully designed Mac application. From Panic, the developers known for thoughtful Mac software, Coda integrates editing, file transfer, terminal, and preview without requiring separate applications or complex configurations.

The <a href="#">code editor</a> provides intelligent text editing. Syntax highlighting for web languages. Code completion assists writing. Editor capabilities match dedicated code editors.

<a href="#">Sites management</a> organizes web projects. Save server connections, local paths, and settings. Switch between projects cleanly. Site organization scales to many projects.

The <a href="#">built-in FTP/SFTP</a> handles file transfer directly. Upload files without separate clients. Edit remote files directly when preferred. Integrated transfer simplifies deployment.

<a href="#">Preview pane</a> shows rendered pages alongside code. See changes without browser switching. Preview updates as you edit. Side-by-side viewing accelerates development.

The <a href="#">built-in terminal</a> provides command-line access. Run commands without leaving Coda. SSH connections to servers. Terminal integration suits development workflows.

<a href="#">CSS editing tools</a> visualize stylesheet changes. Color pickers and value adjusters. See CSS effects immediately. Visual CSS tools accelerate styling.

The <a href="#">publishing workflow</a> deploys sites smoothly. Upload changed files automatically. Sync local and remote directories. Publishing automation handles deployment.

<a href="#">Split editing</a> displays multiple files simultaneously. Reference one file while editing another. Efficient use of screen space. Split views suit multi-file editing.
